Spiking Neural Network 기반 다변량 시계열 예측, 자율주행 파이프라인, 실시간 객체 탐지와 AI Agent 시스템에 이르기까지 <br>

### 🚀 프로젝트 및 경험

**1. 다변량 시계열 예측을 위한 채널 유사성 기반 스파이크 인코딩 기법 연구**

 1) 시계열 데이터 내 다중 채널 간 관계를 유사성을 기반으로 모델링 <br>
 • Spiking Neural Network 기반 시계열 예측 선행 연구의 채널 관계 미활용 문제 인식 <br>
 • 어텐션 메커니즘 기반 채널 관계 모델링을 통해 다변량 시계열 내 채널 관계 정량화

 2) SNN에 필수적인 스파이크 인코딩 단계에 결합하여 최대 11.1%의 예측 정확도 개선 <br>
 • STE 기반 채널 유사도 정보를 스파이크 형태로 변환 및 스파이크 인코딩에 결합 <br>
 • 채널 정보의 결합으로 기존 스파이크 인코딩 방법의 정보 손실 문제 해결

<br>

**2. CARLA 기반 시뮬레이션 환경에서 인지-계획-제어 통합 자율주행 파이프라인 개발**

 1) 장애물 회피 및 조향 안정성 향상을 고려한 로컬 경로 생성 로직 설계 <br>
 • 전방의 다중 장애물 분포 기반으로 회피 방향을 판단해 안전한 주행 경로 생성 <br>
 • Look-ahead 거리 및 곡률 파라미터 조정을 통해 회전 구간에서 조향 안정성 향상
 
 2) 자율주행 차량의 주행 과정에서 생성되는 로컬 경로 시각화 <br>
 • 로컬 경로를 Path 메시지로 발행, RViz에서 주행 차량의 실시간 경로 추종 과정 확인 <br>
 • 플래너·제어 모듈 디버깅 및 주행 거동 검증에 활용

<br>

**3. 영상 데이터에 대한 고정밀·저지연 2D 객체 탐지 모델 구현 및 성능 검증**

 1) ADAS의 고정밀·저지연 요구조건을 만족하기 위해 YOLO11 아키텍처의 규모 별 평가 및 분석 <br>
 • 모델 규모 별 추론 속도와 정확도 간 트레이드오프를 정량적으로 비교 <br>
 • ADAS 실시간 제약 조건 충족 가능한 최적 모델로 YOLO11-Large 도출

 2) mAP@50-95 기준 0.887 달성 및 추론 과정에서 목표치 대비 처리 속도 7.5배 개선 <br>
 • TensorRT 기반 FP16 최적화 적용, 정확도 유지 상태에서 실시간 처리 수준 추론 속도 확보 <br>
 • GPU 환경 기준 초당 150 프레임 처리 성능 달성

<br>

**4. AI Agent 기반 민원 부서 추천·악성 민원 필터링 시스템 개발**

 1) 민원 내용을 수집·분석하여 배정 담당자에게 추천 부서를 제시하는 End-to-End 파이프라인 구현 <br>
 • LLM 임베딩과 VectorDB 유사도 검색을 활용, 과거 민원 기반 부서 추천 기능 구현 <br>
 • 대구 달서구 온라인 민원 데이터 대상으로 Zero-shot 정확도 78.64% 달성

 2) 민원 담당자가 악성 민원에 직접적으로 노출되지 않도록 심리적 완충 기능 구현 <br>
 • 욕설과 희롱, 비꼼 등 부정적 내용을 사전에 감지 후 일시적 흐림 처리 <br>
 • 중복 민원 이력 파악 후 사전 차단을 통해 민원 담당자의 업무 효율성 향상
